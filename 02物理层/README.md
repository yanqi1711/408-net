如何区分曼彻斯特编码和差分曼彻斯特编码？<br>
两种编码都是“中必变”，如果中间跳变方向和二进制能对应就是曼彻斯特编码；如果虚线处的跳变能和二进制对应就是差分曼彻斯特编码，同时还有一段开始的虚线<br>
